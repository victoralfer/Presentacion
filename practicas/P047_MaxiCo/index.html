Selectores

¿Qué son?

En CSS, un selector es una parte de la regla CSS que indica a qué elementos HTML se aplicarán los estilos definidos en esa regla. Esencialmente, es la forma de especificar qué elementos de la página web se verán afectados por las reglas CSS.

Box-Model

¿Qué aporta?

El modelo de caja (box model) de CSS es un concepto fundamental para el diseño web, que describe cómo se representan los elementos HTML en una página como cajas rectangulares. Cada elemento tiene cuatro áreas principales: contenido, relleno, borde y margen.

Listas

¿Cuál es su uso?

En CSS, las listas se pueden personalizar usando diversas propiedades para modificar su apariencia, como el tipo de marcador, la posición y la imagen utilizada. Se pueden aplicar estilos a listas ordenadas 'ol' y desordenadas 'ul'.

FlexBox

¿Para qué sirve?

Flexbox en CSS es un modelo de diseño unidimensional que facilita la alineación y distribución de elementos dentro de un contenedor. Permite que los elementos se expandan o contraigan para adaptarse al espacio disponible, creando diseños flexibles y responsivos.
